  • Explicação sobre UFW, Firewall nativo do Ubuntu

    A UFW é o serviço padrão de firewall para Ubuntu. A UFW foi desenvolvida para conectar facilmente com configurações iptable de firewall. Este serviço dá ao usuário uma maneira conveniente de criar uma firewall baseada no host tanto de ipv4 e ipv6. A UFW vem pré-instalada no ubuntu, mas ela é desabilitada por padrão e é feita para ser acessada pelo terminal, no entanto é possível instalar uma interface gráfica para ele.

  • Passos para configurar of UFW:

    1. $ sudo nano /etc/default/ufw ## Verificar IPV6 = yes, caso não for, altere.
    2. $ sudo ufw disable && sudo ufw enable ## Desliga, e Ligar o Firewall para aplicar o ipv6 caso não aplicou alguma modificação
    3. $ sudo ufw allow ssh || sudo ufw allow 22/tcp ## Ambos tem a mesma função de permitirem conexões por SSH
    4. Comandos Para Negar Conexões:
    5. Bloquear todos os ips de SSH:
    6. $ sudo ufw default deny incoming ## Define o firewall para negar todas as conexões de entrada
    7. Bloquear ip especifico:
    8. $ sudo ufw deny from 203.0.113.4 ## Define um ip especifico para negar,o ip deve ser trocado pelo ip real que você quer bloquear
    9. Comandos Para Permitir Conexões:
    10. <$ sudo ufw allow www ## Permite acesso a todas as páginas www
    11. $ sudo ufw allow 80/tcp ## Para permitir acesso a porta 80 (permitir conexão http do apache e nginx)
    12. $ sudo ufw allow ftp || sudo ufw allow 21/tcp ## Para permitir conexões FTP
